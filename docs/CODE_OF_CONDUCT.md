# Code of Conduct

casanovoutils adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. All contributors are expected to uphold these standards.

To report a violation, please contact the maintainers by opening a private issue on [GitHub](https://github.com/Noble-Lab/casanovoutils/issues) or emailing the Noble Lab directly.
